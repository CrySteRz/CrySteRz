## Featured Project

### [Ravalink](https://github.com/CrySteRz/ravalink)

A scalable Lavalink alternative built in **Rust** with **Kafka**-based message passing. Architected as three independent services with a shared message contract:

| Repository | Role |
|---|---|
| [`ravalink`](https://github.com/CrySteRz/ravalink) | Core audio processing service |
| [`ravalink-lib`](https://github.com/CrySteRz/ravalink-lib) | Rust client library for bot integration |
| [`ravalink-interconnect`](https://github.com/CrySteRz/ravalink-interconnect) | Kafka message schema specification |
| [`ravalink-bot-example`](https://github.com/CrySteRz/ravalink-bot-example) | Example Discord bot implementation |
